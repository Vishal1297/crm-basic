# crm-basic

A customer relationship management app built using Go and Fiber


## Setup Guide

Install Go - [go.dev/doc](https://go.dev/doc/install)

Install Sqlite3 - [www.sqlite.org](https://www.sqlite.org/download.html)


## Commands To Run

`cd crm-basic`

`go mod tidy`

`go build main.go`

`go run main.go`
